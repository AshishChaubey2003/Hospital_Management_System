# 🏥 Hospital Management System (Oracle SQL Project)

## 📌 Overview
This project is a relational database system developed using **Oracle SQL** and **PL/SQL** to manage patients, doctors, appointments, and billing efficiently in a hospital environment.

## ⚙️ Features
- Created fully normalized tables with relationships and constraints.
- Automated billing process using **Stored Procedures** and **Triggers**.
- Implemented **Views** for quick daily appointment and billing reports.
- Optimized queries using indexing, joins, and execution plan analysis.
- Applied **transaction control (COMMIT/ROLLBACK)** for data consistency.
- Ensured security using **GRANT** and **REVOKE** for role-based access.

## 🧠 Technologies Used
Oracle SQL | PL/SQL | Database Design | Query Optimization | Data Integrity | SQL Developer Tool
